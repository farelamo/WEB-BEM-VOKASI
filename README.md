PERSIAPAN 

1. Bikin Database 
```
Nama Database : bem_vokasi_ub
```

2. Generate Key
```
php artisan key:generate
```

3. NPM 
```
npm install
```

```
npm ci
```

```
npm run dev
```
4. Install Debugbar
```
composer require barryvdh/laravel-debugbar
```

5. Install Sweet Alert
```
composer require realrashid/sweet-alert
```

6. Run Migrate Database
```
php artisan migrate
```

7. Run Migrate Database
```
php artisan db:seed
```

8. Run Project
```
php artisan serve
```
